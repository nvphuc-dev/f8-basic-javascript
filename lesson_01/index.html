Toán tử ++ --:

Prefix (Tiền tố, nó đúng như những gì bạn nghĩ)

Việc 1: + 1 cho a, a = a +1 => a = 5 (trường hợp ++)

Việc 2: Trả về a sau khi được + 1

Ví dụ (Prefix)

let a = 4;

++a; // 5

--a; // 3

Postfix (Hậu tố, cách tính toán khác)

Việc 1: copy a ra 1 biến gọi là `a copy`, `a copy` = 4

Việc 2: Trừ 1 của a, a = a - 1 => a = 3

Việc 3: Trả về `a copy`

Tương tự thực hiện cho a++

Ví dụ (Postfix)

let a = 4;

a++; // 4

a--; // 4

let number = 6;

let output = number++ * 2 - --number * 2;

7 * 2 - 7 * 2 => output = 0

Mảng (JavaScript Arrays)

Tạo mảng

Mảng là một biến đặc biệt, có thể chứa nhiều hơn một giá trị:

Syntax:

const array_name = [item1, item2, ...];

Ví dụ

const cars = ["Saab", "Volvo", "BMW"];

const cars = new Array("Saab", "Volvo", "BMW");

Mảng có thể chứa tất cả các loại dữ liệu.

Truy xuất mảng

Ví dụ

const cars = ["Saab", "Volvo", "BMW"];

(cars.length); // 3 (độ dài mảng)

(cars[0]); // Saab (lấy phần tử theo index)

Cách nhận biết một mảng

Vấn đề là toán tử JavaScript typeof trả về "object":

Giải pháp 1: Để giải quyết vấn đề này, ECMAScript 5 (JavaScript 2009) đã định nghĩa một phương thức mới Array.isArray()

const cars = ["Saab", "Volvo", "BMW"];

let type = typeof cars;

(Array.isArray(cars));

Giải pháp 2: Toán tử instanceof trả về true nếu một đối tượng được tạo bởi một hàm tạo đã cho

const cars = ["Saab", "Volvo", "BMW"];

(cars instanceof Array);

(cars.shift()); // [Saab]

JavaScript Array unshift()

Phương thức unshift() thêm 1 hay nhiều phần tử vào đầu mảng và trả về độ dài mới của mảng.

Ví dụ

const cars = ["Saab", "Volvo", "BMW"];

(cars.unshift('Toyota', 'Honda')); // [Toyota, Honda, Saab, Volvo, BMW]

JavaScript Array splice()

Phương thức splice() thêm các phần tử mới vào một mảng.

Ví dụ

const cars = ["Saab", "Volvo", "BMW"];

(cars.splice(1, 0, 'Toyota', 'Honda')); // [Saab, Toyota, Honda, Volvo, BMW]

Tham số đầu tiên (1) xác định vị trí mà các phần tử mới sẽ được thêm vào (spliced in).
Tham số thứ hai (0) xác định số lượng phần tử sẽ bị xóa.
Các tham số còn lại ("Toyota" , "Honda") xác định các phần tử mới sẽ được thêm vào.
Phương thức splice() trả về một mảng với các mục đã bị xóa

JavaScript Array slice()

Phương thức slice() cắt một phần của mảng thành một mảng mới.

Ví dụ

const cars = ["Saab", "Volvo", "BMW"];

(cars2 = cars.slice(1)); // [Volvo, BMW]

Phương thức slice() tạo một mảng mới.
Phương thức slice() không xóa bất kỳ phần tử nào khỏi mảng nguồn.

Ví dụ

const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];

const citrus = fruits.slice(1,3);

(citrus); // [Orange, Lemon]

Phương thức slice() có thể nhận hai đối số như slice(1, 3).
Sau đó, phương thức chọn các phần tử từ đối số bắt đầu và tối đa (nhưng không bao gồm) đối số kết thúc.

Merging (Concatenating) Arrays

Phương thức concat() hợp nhất các mảng lại với nhau.

Ví dụ

const myGirls = ["Cecilie", "Lone"];

const myBoys = ["Emil", "Tobias", "Linus"];

const others = ["Kenvin", "Teresa"];

(myChildren = myGirls.concat(myBoys)); // [Cecilie, Lone, Emil, Tobias, Linus]

(myFamily = myGirls.concat(myBoys, others)); // [Cecilie, Lone, Emil, Tobias, Linus, Kenvin, Teresa]

Phương thức concat() không thay đổi các mảng hiện có. Nó luôn trả về một mảng mới.

Phương thức concat() có thể nhận nhiều đối số.

Hàm (JavaScript Functions)

Hàm JavaScript là một khối mã được thiết kế để thực hiện một tác vụ cụ thể.

Một hàm JavaScript được thực thi khi "cái gì đó" gọi nó (calls it).

Ví dụ

// Hàm tính tích của p1 và p2

function myFunction(p1, p2) {

    return p1 * p2;

}

Cú pháp hàm trong Javascript

Syntax:

function function_name(parameter1, parameter2, parameter3) {

    // mã được thực thi

}

Tham số trong hàm (parameter)

Các tham số của hàm được liệt kê bên trong dấu ngoặc đơn () trong định nghĩa hàm.

Các đối số của hàm là các giá trị mà hàm nhận được khi nó được gọi.

Bên trong hàm, các đối số (các tham số) hoạt động như các biến cục bộ.

Ví dụ

function writeLog() {

    for(let param of arguments) {

        console.log(param);

    }

}

(writeLog('Log 1', 'Log 2', 'Log 3');) // Log 1 \n Log 2 \n Log 3

Return trong function (Function Return)

Khi JavaScript đạt đến câu lệnh trả về, hàm sẽ ngừng thực thi.

Nếu hàm được gọi từ một câu lệnh, JavaScript sẽ "quay lại" để thực thi mã sau câu lệnh gọi.

Các hàm thường tính toán một giá trị trả về. Giá trị trả về được "trả lại" cho "người gọi.

Ví dụ

// Tính tích 2 số và trả kết quả

function myFunction(a, b) {

    return a * b; // Hàm trả về tích của a và b

}

let x = myFunction(4, 3); // Gọi được hàm tính tích 2 số, giá trị trả về cho người gọi x = 12

Return có thể trả về bất kỳ kiểu dữ liệu nào.

Function đặt trùng tên thì function sau ghi đè lên function trước.

Biến trong function chỉ chạy trong phạm vi function đó.

Function có thể chứa function khác.

Nếu function không return thì sẽ trả về undefined.

Các loại Function

Function declartion

Khi bạn tạo ra một function có tên, đó gọi là function declaration

Ví dụ

function handleClick() {

    console.log("clicked")

}

Hàm có thể gọi trước hoặc sau khi được định nghĩa.

Function expression

Cách khai báo khác với function declaration

Ví dụ

const handleClick = function() {

    console.log("clicked")

}


// Cú pháp với ES6 cũng là function expression

const handleClick = () => {

    console.log("clicked")

}

Hàm chỉ gọi thực thi sau khi được định nghĩa.

Chúng ta sử dụng function declaration khi muốn tạo ra function để sử dụng ở bất cứ đâu trong toàn bộ mã code và sử dụng function expression khi function bị giới hạn vùng sử dụng, giúp global scope nhẹ và sạch hơn.

Hoisting

Khái niệm hoisting biểu thị function hay biến có thể được "gọi" ngay từ dòng code đầu tiên, trước khi chúng được khai báo.

Function Declaration có thuộc tính hoisting còn Function Expression thì không.

Ví dụ

handleStuff();


function handleStuff() {

    // do stuff

}

Đoạn code dưới đây sẽ thực thi bình thường khi Function Declaration.

Ví dụ

handleStuff();


const handleStuff = function() {

    // do stuff

}

Đối với Function Expression thì sẽ báo lỗi.

Arrow function

Arrow function là 1 feature của CoffeeScript, giúp việc định nghĩa function trở nên ngắn gọn hơn (từ phiên bản ES6).

Ví dụ

let helloWorld = () => { console.log(‘Hello World’) }

hello = (val) => "Hello " + val; // với các tham số

Nếu funtion không chỉ có 1 dòng code, chúng ta có thể bỏ {}
let HelloWorld = () => console.log('Hello World')

Nếu function chỉ có return statement:
let helloWorld = () => "Hello World";

Với function chỉ có 1 parameter, chúng ta có thẻ bỏ ()
let hello = name => console.log(Hello ${name})

Lệnh rẽ nhánh If, Else

Câu lệnh điều kiện được sử dụng để thực hiện các hành động khác nhau dựa trên các điều kiện khác nhau.

Câu lệnh if

Syntax:

if (condition) {

    // block of code to be executed if the condition is true

}

Ví dụ

if (hour < 18) {

    greeting = "Good day";

}


Kết quả của lời chào sẽ là: Good day

Câu lệnh if, else

Syntax:

if (condition) {

    // block of code to be executed if the condition is true

} else {

    // block of code to be executed if the condition is true

}

Ví dụ

if (hour < 18) {

    greeting = "Good day";

} else {

    greeting = "Good evening";

}


Kết quả của lời chào sẽ là: Good day

Ví dụ

if (time < 10) {

    greeting = "Good morning";

} else if(time < 20) {

    greeting = "Good day";

} else {

    greeting = "Good evening";

}


Kết quả của lời chào sẽ là: Good day

Lệnh rẽ nhánh Switch

Câu lệnh chuyển đổi được sử dụng để thực hiện các hành động khác nhau dựa trên các điều kiện khác nhau.

Syntax:

switch(expression) {

case x:

    // code block

    break;

case y:

    // code block

    break;

default:

    // code block

}

Dùng if, else khi xác định tính đúng sai, switch khi biết trước điều kiện và số điều kiện nên lớn hơn 3.

Toán tử 3 ngôi (Ternary operator)

Syntax:

variable_name = (condition) ? value1 : value2;

Ví dụ

let exp = 1;

let salary = exp > 3 ? 1000 : 500;

console.log(salary) // 500

Vòng lặp (Loop)

Vòng lặp for

Các vòng lặp có thể thực thi một khối mã nhiều lần.

Syntax:

for (điều kiện 1; điều kiện 2; điều kiện 3) {

    // code block to be executed

}

Ví dụ

for(var i = 1; i <= 10; i++){

    console.log(i);

}

Vòng lặp for/in

Câu lệnh for in trong JavaScript lặp qua các thuộc tính của một Đối tượng:

Syntax:

for (key in object) {

    // code block to be executed

}

Ví dụ

let myInfo = {

    name: 'Kenvin Nguyen',

    age: 35,

    address: 'Quang Ngai'

}

for(let key in myInfo){

    console.log(myInfo[key]);

}

Vòng lặp for/of

Câu lệnh for of của JavaScript lặp qua các giá trị của một đối tượng có thể lặp lại.
Nó cho phép bạn lặp qua các cấu trúc dữ liệu có thể lặp lại như Mảng, Chuỗi, Bản đồ, NodeLists, v.v.:

Syntax:

for (variable of iterable) {

    // code block to be executed

}

Ví dụ

let myInfo = {

    name: 'Kenvin Nguyen',

    age: 35,

    address: 'Quang Ngai'

}

for(let value of Object.values(myInfo)){

    console.log(value);

}

Vòng lặp while

The while loop loops through a block of code as long as a specified condition is true.

Syntax:

while (condition) {

    // code block to be executed

}

Ví dụ

while (i < 10) {

    text += "The number is " + i;

    i++;

}

Vòng lặp do/while

The do while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.

Syntax:

do {

    // code block to be executed

}

while (condition);

Ví dụ

do {

    text += "The number is " + i;

    i++;

}

while (i < 10);

Break & Continue trong vòng lặp

The break statement "jumps out" of a loop.

The continue statement "jumps over" one iteration in the loop.

Ví dụ

for(let i = 0; i < 10; i++){

    if(i % 2 !== 0 ){

        continue;

    }

    console.log(i); //In các sổ chẵn

}

Vòng lặp lồng nhau

Ví dụ

let array = [

    [1, 2],

    [3, 4],

    [5, 6]

];

for(let i = 0; i < array.length; i++){

    for(let j = 0; j < array[i].length; j++){

        console.log(array[i][j]);

    }

}

Làm việc với mảng nâng cao

forEach()

Phương thức forEach() gọi một hàm (hàm gọi lại) một lần cho mỗi phần tử mảng.

Ví dụ

const numbers = [45, 4, 9, 16, 25];

let txt = "";

numbers.forEach(myFunction);

function myFunction(value, index, array) {

    txt += value + "< br >";

}

every()

Phương thức every() kiểm tra xem tất cả các giá trị mảng có vượt qua bài kiểm tra hay không.

Ví dụ

const numbers = [45, 4, 9, 16, 25];

let allOver18 = numbers.every(myFunction);

function myFunction(value, index, array) {

    return value > 18;

}

Ví dụ này kiểm tra xem tất cả các giá trị mảng có lớn hơn 18 hay không?

some()

Phương thức some() kiểm tra xem một số giá trị mảng có vượt qua bài kiểm tra hay không.

Ví dụ

const numbers = [45, 4, 9, 16, 25];

let someOver18 = numbers.some(myFunction);

function myFunction(value, index, array) {

    return value > 18;

}

Ví dụ này kiểm tra xem một số giá trị mảng có lớn hơn 18 không?

find()

Phương thức find() trả về giá trị của phần tử mảng đầu tiên vượt qua hàm kiểm tra.

Ví dụ

const numbers = [45, 4, 9, 16, 25];

let first = numbers.find(myFunction);

function myFunction(value, index, array) {

    return value > 18;

}

Ví dụ này tìm (trả về giá trị của) phần tử đầu tiên lớn hơn 18

find() là một tính năng ES6 (JavaScript 2015).

map()

Phương thức map() tạo một mảng mới bằng cách thực hiện một hàm trên từng phần tử mảng.
Phương thức map() không thực thi hàm đối với các phần tử mảng không có giá trị.
Phương thức map() không thay đổi mảng ban đầu.

Chỉnh sửa hay thổi đổi 1 array

Ví dụ

const numbers1 = [45, 4, 9, 16, 25];

const numbers2 = numbers1.map(myFunction);

function myFunction(value, index, array) {

    return value * 2;

}

Ví dụ

const numbers1 = [45, 4, 9, 16, 25];

const numbers2 = numbers1.map(myFunction);

function myFunction(value) {

    return value * 2;

}

reduce()

Phương thức reduce() chạy một hàm trên từng phần tử mảng để tạo (rút gọn thành) một giá trị đơn.

Phương thức reduce() hoạt động từ trái sang phải trong mảng. Xem thêm lessRight().

The reduce() method does not reduce the original array.

Ví dụ: Flat - 'Làm phẳng' mảng từ Depth array - 'Mảng sâu'

let depthArray = [1, 2, [3, 4], 5, 6, [7, 8, 9]];

let flatArray = depthArray.reduce(function(flatOutput, depthItem){

    return flatOutput.concat(depthItem);

}, []);

console.log(flatArray); // [1, 2, 3, 4, 5, 6, 7, 8, 9]

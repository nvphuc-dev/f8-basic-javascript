Giới thiệu về Javascript

Javascript có thể làm được gì?

  • JavaScript là ngôn ngữ lập trình phổ biến nhất thế giới.
  • JavaScript là ngôn ngữ lập trình của Web.
  • JavaScript rất dễ học.
  • Hướng dẫn này sẽ dạy bạn JavaScript từ cơ bản nhất.
Cài đặt môi trường

Cài đặt các công cụ cần thiết:

  • Trình duyệt Google Chrome (Mozila Firefox,...)
  • Visual Studio Code + Live server extension
Biến trong JavaScript

4 Cách Khai báo Biến JavaScript:

  • Sử dụng var
  • Sử dụng let
  • Sử dụng const
  • Không sử dụng gì

Ví dụ

var x = 5;

var y = 6;

var z = x + y;

Toán tử

1. Toán tử số học - Arithmetic:

Toán Tử Mô Tả
+ Cộng
- Trừ
* Nhân
** Lũy thừa
/ Chia
% Chia lấy số dư
++ Tăng 1 giá trị số
-- Giảm 1 giá trị số

Ví dụ

let a = 4;

let b = 24;

let x = 4 + 24; // 28

let z = b - a; // 20

let e = b * a; // 96

let c = b / a; // 6

let d = b % a; // 0

a++;

let h = a; // 5

b--;

let t = b; // 23

let f = 5;

2. Toán tử ++ --:

Prefix (Tiền tố, nó đúng như những gì bạn nghĩ)

Việc 1: + 1 cho a, a = a +1 => a = 5 (trường hợp ++)

Việc 2: Trả về a sau khi được + 1

Ví dụ (Prefix)

let a = 4;

++a; // 5

--a; // 3

Postfix (Hậu tố, cách tính toán khác)

Việc 1: copy a ra 1 biến gọi là `a copy`, `a copy` = 4

Việc 2: Trừ 1 của a, a = a - 1 => a = 3

Việc 3: Trả về `a copy`

Tương tự thực hiện cho a++

Ví dụ (Postfix)

let a = 4;

a++; // 4

a--; // 4

let number = 6;

let output = number++ * 2 - --number * 2;

7 * 2 - 7 * 2 => output = 0

3. Toán tử gán:

Toán Tử Ví dụ Tương dương
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
<<= x <<= y x = x << y
>>= x >>= y x = x >> y
>>>= x >>>= y x = x >>> y
&= x &= y x = x & y
^= x ^= y x = x ^ y
|= x |= y x = x | y
**= x **= y x = x ** y

The **= operator is a part of ECMAScript 2016.
